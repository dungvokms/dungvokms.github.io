[Issue] [IntelliJ] Import Gradle projects and got errors

January 5, 2019

Suppose that you want to import a Gradle projects from Github, Bitbucket, Gitlab and you have the below error when importing to IntelliJ

No cached version available for offline mode

So to fix this issue, go to IntelliJ settings/preferences, search for Gradle, and uncheck the Offline work

Re-import project and it should work!
